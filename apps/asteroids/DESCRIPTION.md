# 🚀 Asteroids для Lilka

Порт класичної аркади Asteroids** (Atari, 1979) на консоль Lilka


## ✨ Можливості

- Інерційна фізика корабля в стилі оригіналу
- Астероїди спавняться з-за країв екрана
- Три режими звуку: без звуку, базер, динамік
- Кілька варіантів стартового банера

## 🎮 Керування

| Кнопка      | Дія                 |
|-------------|---------------------|
| UP / DOWN   | Прискорення вперед / назад |
| LEFT / RIGHT| Поворот корабля     |
| A           | Стрільба            |
| SELECT      | Режим звуку (в меню)|
| START       | Старт / вихід       |
