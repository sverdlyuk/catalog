# ⏱️ Timer for Lilka

Простий та зручний таймер для портативної консолі **Lilka (ESP32-S3)** з підтримкою керування кнопками та звукового сповіщення.

## ✨ Можливості

- Встановлення часу (години, хвилини, секунди)
- Запуск / пауза таймера
- Зворотний відлік у реальному часі
- Звукове сповіщення після завершення
- Інтуїтивне керування кнопками
- Мінімалістичний інтерфейс

## 🎮 Керування

| Кнопка | Дія |
|--------|-----|
| ⬅️ / ➡️ | Перемикання між полями (год / хв / сек) |
| ⬆️ / ⬇️ | Зміна значення |
| 🅰️ | Старт / Пауза / Скидання |
| 🅱️ | Вихід з програми |

## 🖥️ Інтерфейс

- Формат часу: `HH:MM:SS`
- Активне поле підсвічується
- Динамічні підказки кнопки **A**:
  - `Start` — запуск
  - `Stop` — пауза
  - `Reset` — після завершення

## 🔊 Звук

Після завершення таймера відтворюється мелодія через бузер.

## ⚙️ Як це працює

- Час задається у трьох полях: години, хвилини, секунди
- При запуску обчислюється загальна кількість секунд
- Таймер віднімає `delta` (час між кадрами)
- При досягненні нуля:
  - таймер зупиняється
  - запускається мелодія
  - переходить у стан `finished`
