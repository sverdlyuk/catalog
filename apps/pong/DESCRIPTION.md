🏓 Pong для Lilka

Класична аркадна гра Pong, портована на консоль **Lilka** (mJS). Один гравець проти простого CPU-суперника. 🎮

## ✨ Можливості

- 🕹️ Керування лівою ракеткою через D-pad
- 🤖 CPU-суперник зі слідкуванням за м'ячем
- ⚡ Прискорення м'яча після кожного удару по ракетці
- 🎯 Підрахунок рахунку для обох сторін
- 🎨 Мінімалістична графіка: бірюзова й червона ракетки, пунктирна центральна лінія

## 🎮 Керування

| Кнопка       | Дія                       |
|--------------|---------------------------|
| ⬆️ UP        | Рух ракетки вгору         |
| ⬇️ DOWN      | Рух ракетки вниз          |
| 🅰️ A         | Вийти з гри               |

## 🧠 Як це працює

- М'яч відбивається від верхньої та нижньої стінок
- Після зіткнення з ракеткою горизонтальна швидкість зростає на `0.3` — гра стає складнішою з часом
- Якщо м'яч вилетів за ліву чи праву межу — суперник отримує очко, а м'яч повертається в центр
- CPU рухає праву ракетку до позиції м'яча зі швидкістю `3` пікселі за кадр (трохи повільніше за гравця — щоб було чесно 😉)

## 🚀 Запуск

Скопіювати `pong.js` у каталог застосунків на SD-картці Lilka та запустити через меню консолі.
