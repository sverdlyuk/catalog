# BMP280 для Лілки

Програма на Lua для відображення температури та тиску з датчика BMP280 на консолі [Лілка](https://lilka.dev).

## Підключення

| BMP280 | Лілка |
|--------|-------|
| VCC    | 3.3V  |
| GND    | GND   |
| SCL    | пін 14 |
| SDA    | пін 13 |

> Якщо датчик не знаходиться — спробуй адресу `0x77` (SDO підключено до VCC)

## Можливості

- Автоматичний вимір кожні 60 секунд
- Журнал до 100 вимірів
- Графік зміни температури з прокруткою
- Debug екран з детальними проміжними значеннями формул компенсації

## Кнопки

**Головний екран**
- `A` — виміряти зараз
- `B` — вихід
- `C` — журнал вимірів
- `D` — графік температури

**Графік**
- `LEFT / RIGHT` — прокрутка по часу
- `B` — назад

**Журнал**
- `UP / DOWN` — прокрутка списку
- `A` — детальний debug виміру
- `B` — назад

## Технічні деталі

Датчик працює у **forced mode** — вмикається лише на момент виміру (~13 мс) і одразу переходить у сон. Це мінімізує самонагрів і дає точніші показники температури.

Формули компенсації — 32-bit integer з даташиту Bosch (розділ 8.2). Бітові зсуви `>>` замінені на `//` (floor division) через особливість Lua: оператор `>>` є логічним (unsigned), що дає хибний результат на від'ємних проміжних значеннях.
