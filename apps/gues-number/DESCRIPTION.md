# Вгадай число для Lilka

Простенька, але захоплива гра для Lilka, де потрібно вгадати число від 0 до 100.

## 🎮 Як грати

- **UP/DOWN** — змінити число
- **A** — підтвердити здогадку
- **B** — вийти з гри

## 🎯 Геймплей

1. На екрані відображається число, яке ти обираєш за допомогою кнопок UP/DOWN
2. Натисни **A**, щоб зробити здогадку
3. Гра подасть підказку: **"Більше!"** або **"Менше!"**
4. Продовжуй вгадувати, поки не знайдеш правильну відповідь
5. Після перемоги програма покаже:
   - Загадане число
   - Кількість спроб, які знадобилися
   - Цікавий факт про це число
